1. What is the difference between var, let, and const?
Answer:
var, let, and const are used to declare variables in JavaScript, but they work in slightly different ways.

var is the old way of declaring variables. It can be re-declared and updated, and it is function-scoped.
let is a modern way to declare variables. It can be updated but cannot be re-declared in the same scope. It is block-scoped, which means 
it only worksinside the block where it is declared.
const is used for variables whose value should not change. It cannot be re-declared or updated after it is assigned.

2.What is the spread operator (...)?
Answer:
The spread operator (...) is used to expand elements of an array or object. It allows us to copy or combine arrays and objects easily.
For example, we can copy an array or merge two arrays using the spread operator.

3.What is the difference between map(), filter(), and forEach()?
Answer:
map(), filter(), and forEach() are array methods used to work with elements in an array.

map() is used when we want to create a new array by changing each element.
filter() is used to create a new array with elements that match a condition.
forEach() is used to loop through an array and perform an action, but it does not return a new array.

4.What is an arrow function?
Answer:
An arrow function is a shorter way to write functions in JavaScript. It uses the => symbol.
It makes the code more simple and readable.Arrow functions are commonly used in modern JavaScript programs.

5.What are template literals?
Answer:
Template literals are a way to create strings in JavaScript using backticks ( ). They allow us to insert variables or
expressions inside a string.We use ${} to include variables.Template literals make string writing easier and more flexible.
